### Instalación:
```bash
git clone "https://github.com/migvelf/automatizacion.git"
cd automatizacion
pip install -r requirements.txt
source .comandos.sh
En create.py usar sus credenciales de GitHub user=Usuario password=Contraseña.
Cambiar la ruta de directorios  '/ruta/del/proyecto'
```

### Uso:
```bash
Para ejecutar el script 'create <nombre del proyecto>'
```
